Project NANPRE v2.2

追加機能
- 連結9×9盤面を指でドラッグし、マス単位で吸着
- 重ねたマスを共有
- 12×12（3×4ブロック）
- 16×16（4×4ブロック）
- 写真を選択
- 写真の拡大、左右移動、上下移動、回転補正
- グリッドに合わせた写真を盤面へ透過表示
- 選択マスの拡大表示
- 写真を見ながら数字を手入力
- 自動解答、矛盾チェック、灰色マス合計、途中保存
