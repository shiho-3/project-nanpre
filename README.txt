Project NANPRE（PWA版）

このZIPは、そのままWeb公開できる完成パッケージです。

中身
- index.html：アプリ本体
- manifest.webmanifest：ホーム画面追加用設定
- service-worker.js：オフライン利用用
- icons：アプリアイコン

主な機能
- 通常9×9
- 4連、7連、12連テンプレート
- 盤面を追加して変則配置に対応
- 重なったマスを自動共有
- 入力矛盾チェック
- 自動解答
- 灰色マスの指定と合計表示
- 端末内の途中保存
- ホーム画面追加
- 一度開いた後のオフライン利用

重要
PWAとしてSafariから使うには、GitHub PagesなどのHTTPSサイトへ置く必要があります。
ZIPをiPhoneに保存しただけでは、PWA機能は有効になりません。
